Website Evaluations


React (www.reactjs.org)

React is a JavaScript library for building user interfaces

  • Contrast: Excellent contrast throughout the page, only changing colors and themes when needing to highlight something
  • Repetition: Key themes are reused throught the site, in a clear and aestheic manner
  • Alignment: Everything is aligned perfectly and I could not find anything that could use improvement
  • Proximity: Everything is easily accessible and grouped accordingly

Demand Driven Institute (www.demanddriveninstitute.com)

The company Demand Driven Institute has a model that the company seeks to implement into other businesses

  • Contrast: This website has so much contrast to the point where it is essentially unusable. The bright yellow,green and red would give most users a migrane
  • Repetition: The repetition is so overused that the box icons with colors take over the page
  • Alignment: The alignment is overall well done except for the occasional random icon scaterred throughout the screen
  • Proximity: There seems to be no pattern for grouping elements
